## moodle forum delete

Delete a post

### Synopsis

Delete a post by its ID. Deleting the first post in a discussion deletes the entire discussion.

```
moodle forum delete <post-id> [flags]
```

### Examples

```
  # Delete a post
  moodle forum delete 200
```
